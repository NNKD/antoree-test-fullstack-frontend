## TechStack
- **React JS**
- **Tailwind CSS (v3.4.17)**

---

## How To Use
1. Install dependencies
    ```shell 
    npm install
    ```
2. Setup environment variables

   > Copy **.env.example** to **.env** and update the values

3. Run the project
    ```shell
    npm run dev
    ```
